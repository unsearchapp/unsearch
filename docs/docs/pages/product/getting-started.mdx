# Get started

To begin using Unsearch, follow these simple steps:
::::steps

### Install the Unsearch Extension

Start by installing the Unsearch extension for your preferred browser. Unsearch supports the following browsers:

- [Chrome](https://chromewebstore.google.com/detail/unsearch/hcalfepkjgcohdhnjhdpbnifeopodkkh)
- [Firefox](https://addons.mozilla.org/es/firefox/addon/unsearch/)
- [Edge]()

### Create an Account

After installing the extension, create an account with Unsearch. This will allow you to sync and access your browsing data.

### Access Your Dashboard

Once your account is set up, you can access your synced data on [dashboard.unsearch.app](https://dashboard.unsearch.app) from any device. This gives you the flexibility to manage your bookmarks, tabs, and search history wherever you are.
